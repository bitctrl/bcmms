Chapter 9. Systray

In the window Systray (see Figure 9.1, “Window Options- Systray”) the user can adjust, which functions should be available via the bitcontrol® Digital TV Link context menu (see Chapter 10, bitcontrol® Digital TV Link control).

Figure 9.1. Window Options- Systray


Show Icon in Taskbar Status Area: If this option is active, the systray icon will be shown in the footer of the desktop. This option is active by default.

Arrange by: In this field, the user can configure which criteria should be used for channel sort. Provider, Alphabetically and Frequency are selectable. The option Provider is chosen by default.

Programs Menu: The user can configure which available channels should be shown by activation of the respective checkbox(es). The choices are:

  • Show TV Programs
  • Show Radio Programs
  • Show Encrypted Programs

    Note

    Special rights or requirements are needed to show encrypted programs!
  • Show Other Services

Show Filters Menu: If this checkbox is active, the entry Filter appears in the bitcontrol® Digital TV Link context menu. The bitcontrol® Digital TV Link properties can be opened via the context menu. The checkbox is not active by default.

Show Bouquets: If this checkbox is deactivated, the EPG data aren't available anymore. The checkbox is active by default.

Show Program History: If this checkbox is deactivated, the channel history (see Chapter 10, bitcontrol® Digital TV Link control won't be shown anymore in the context menu anymore. The checkbox is active by default.

History Length: Maximal channel amount of the channel history (see Chapter 10, bitcontrol® Digital TV Link control

The button Clear behind the field History Length deletes the channel history in the bitcontrol® Digital TV Link contect menu.
