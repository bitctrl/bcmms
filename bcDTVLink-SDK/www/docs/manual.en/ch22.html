Chapter 22. Plug-in Traficon VIPT

Traficon encoder devices are supported as input sources by bitcontrol® Digital TV Link via the plug-in Traficon VIPT.

The plug-in can be started in the main window (see Figure 22.1, “Plug-in Traficon VIPT - Main window”) with the help of the Start button and can be stopped with the Stop button.

Figure 22.1. Plug-in Traficon VIPT - Main window


Note

The functionalities of the plug-in Traficon VIPT are in essence similar to the RTSP Receiver plug-in functions. You can find more information about the configuration possibilities in Chapter 21, Plug-in RTSP Receiver.
