The connected Set Top Box can be configured via the dialogues of the respective entries DBox2, DreamBox and ReelBox. All three STB types will be configured according to the same procedure. Therefore the user can follow the described example of configuring a DreamBox.

16.1. Preferences

The entry Preferences opens a window where the user can enter or change settings of his respective STB.

Figure 16.1. Window Preferences


Address: The IP address of the Set Top Box must be entered in this text area.

STB Model: The user has to select his specific Set Top Box type in this selection field. The following STB types are supported at the moment: DM 500/DM 600, DM 7000, DM 7020, DM 7025, DM 8000.

Streaming Method: In this selection field, the user can define, which streaming method should be used. The options PES and TS are supported.

Channels refresh interval (min): This value indicates how many minutes should pass before the next automatic channel update takes place.

The button Update Channellist behind the text field releases the channel update immediately.

16.2. Channels

The user receives a window in which all available channels are listed via the entry Channels. The list is created automatically out of the Set Top Box data.

Figure 16.2. Available channels of the STB


Radio channels have a table background color, so they can be differed from video channels.

16.2.1. Add and delete channels

To add or delete channels the right mouse button has to be used. It makes a submenu appear.

Figure 16.3. Add or delete channels menu


Depending on the desired action, the user chooses the entry Add Channel to add a channel to the list and/or Remove Channel(s) to delete a channel from the list here. (For the latter case, the desired channel has to be marked first.)

To edit the new entry (e.g. an added channel), the user double clicks on the respective field and enters the data into the text field.

The entry will be assumed with the Apply button.

16.2.2. Bouquets

It is possible to create personal channel bouquets with the bitcontrol® Digital TV Link. More information can be found in Section 13.3.2, “Bouquet creation”.
