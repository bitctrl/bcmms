The entry BDA Tuner opens the area, where several settings for USB TV sticks and/or PCI TV cards with BDA drivers can be configures.

Here a USB TV stick is used to explain the functions. But a PCI TV card works similar with bitcontrol® Digital TV Link, so the functionalities can be adapted.

Figure 13.1. Plug-in BDA Tuner - Main window


The inactive plug-in can be started with the Start button.

The running plug-in can be stopped with the Stop button.

The settings will be confirmed with the Apply button.

The button OK closes the window and the button Cancel aborts the process.

Note

Unavailable functions are greyed out.

13.1. Preferences

The user can select the connected device in the plug-in window Preferences (see Figure 13.2, “Plugin BDA Tuner - Preferences”). There it is also possible to do some general device adjustments.

Figure 13.2. Plugin BDA Tuner - Preferences


Select Tuner: The selection field Select Tuner contains a list of all available devices (PCI cards or USB TV sticks). The one that should be used can be selected here. The device settings will be assumed automatically.

Manual: If this checkbox is active, the device settings can be entered or changed by hand.

DVB System: The DVB system can be selected out of this selection field. The following possibilities are available: DVB-T, DVB-S, DVB-C.

Tuner: The selection field "Select Tuner" contains a list of all available devices (PCI cards or USB TV sticks). The one that should be used can be selected here.

Demodulator: The user can choose the demodulator that should be used in the application out of this field. The choices are: BDA Slip De-Framer, BDA MPE-Filter and possibly device-dependent.

Capture: The user can choose the capture component, which should be used in the application, out of this field. The choices are: BDA Slip De-Framer, BDA MPE-Filter and possibly device-dependent.

DiSEqC via "InputRange": This option is only available in DVB-S systems.

8VSB Modulation for DVB-S2: This option is only available in DVB-S2 systems.

Safe transport stream to the file: If this option is active the transport stream will be recorded. The path to the storage location of the transport stream can be entered in the text field under the option box or selected with the Browse button.

Note

Further storage space will be needed!

The settings will be confirmed by the button Apply.

The OK button closes the window and the Cancel button aborts the process.

Note

Functions that aren't available in the DVB system are greyed out.

13.2. Statistics

The plug-in window Statistics (see Figure 13.3, “Plugin BDA Tuner - Statistic”) contains statistical values of the selected channel.

Figure 13.3. Plugin BDA Tuner - Statistic


Locked: If this checkbox is active, channels with lower signal quality will be shown as well. This option is active by default.

Present: If this checkbox is active, channels with good signal quality will be shown. This option is active by default.

Signal Quality: The progress bar Quality shows the intensity of a channel signal in percent. This value should be as high as possible (ca. 100%).

Signal Strength: Shows the received field strength of a selected channel

Total Read (bytes): Shows the Total Read-Parameters for data reception in bytes

Rate (bytes/sec): Shows data reception rate in bytes per second

Extended Information: If this checkbox is active, the underneath table will be filled with more statistical channel information (e.g. bandwidth, frequency, etc.). The checkbox is not active by default.

The settings will be confirmed by the Apply button.

The button OK closes the window and the Cancel button aborts the process.

Note

Unavailable functions are greyed out.

13.3. Channels

Available channels can be searched and adjusted via the plug-in window Channels.

After the first call of the application the window is still empty. A channel seach (see Figure 13.4, “Plug-in BDA Tuner - Search channels”) is absolutely necessary. For this, the user has to select the respective region out of the text field Region and has to click on the Scan button. The search progress will be shown in the progress bar Progress. If a channel was found, the channel appears in the underneath part of the window. Furthermore, the channel quality will be shown in the notifications Level and Quality.

Free Only: If this option is active only unlocked channels will be shown.

Figure 13.4. Plug-in BDA Tuner - Search channels


The search is over when all available channels have been found. The channels can be assumed with the Apply button. If it was successful, the notification colour of the channel list changes from red to green (see Figure 13.5, “Plug-in BDA Tuner - Assumed channels”). Now the channels are available in the system.

Figure 13.5. Plug-in BDA Tuner - Assumed channels


The settings will be confirmed by the Apply button.

The OK button closes the window and the Cancel button aborts the process.

Note

Unavailable functions are greyed out.

13.3.1. Add / remove channel

Channels can be added/removed into/out of the channel list. For this, the user clicks on the selected channel with the right mouse button. The submenu shown in Figure 13.6, “Add/remove channel - Submenu” opens.

Figure 13.6. Add/remove channel - Submenu


Add channel

To add a new channel the user selects the entry Add channel out of the channel list submenu (see Figure 13.6, “Add/remove channel - Submenu”). A new line with red coloured entries will be created in the table. An entry can be edited by double clicking it (see Figure 13.7, “Edit created channel”).

Figure 13.7. Edit created channel


Remove channel

To add a new channel the user selects the entry Remove Channel(s) out of the channel list submenu (see Figure 13.6, “Add/remove channel - Submenu”). The selected channel disappears out of the channel list.

13.3.2. Bouquet creation

It is possible to create personal channel bouquets with the bitcontrol® Digital TV Link. For this the entry Bouquets must be selected out of a channel submenu. A further submenu opens where personal channel bouquets can be created and which also contains all available channels.

Note

If no personal channel bouquets have been created yet, the bouquets submenu contains only the entries Add to... and Add to Favorites (see Figure 13.6, “Add/remove channel - Submenu”).

Create personal channel bouquet

A personal channel bouquet can be created with the bouquets submenu (see Figure 13.6, “Add/remove channel - Submenu”). After selecting the entry Add to... the dialogue shown in Figure 13.8, “Create personal channel bouquet” opens. The name of the personal channel bouquet should be entered in the text field Bouquet Name. The OK button assumes the setting and the new channel bouquet appears as an entry in the bouquets submenu. The process can be aborted with the Cancelbutton.

Figure 13.8. Create personal channel bouquet


Add channel to a created bouquet

To add a channel to a created channel bouquet, the favoured channel bouquet must be selected out of the bouquets submenu. Selected channels will be marked red afterwards (see Figure 13.9, “Channels added to a channel bouquet will be marked red”).

Figure 13.9. Channels added to a channel bouquet will be marked red


Remove channel out of a created bouquet

To remove a channel from a channel bouquet, the respective entry Remove from [Bouquet name] must be selected from the bouquets submenu.

Note

A channel can contain more than one channel bouquet. So it can have more than one entry of Remove from [Bouquet name].

13.3.3. Statistic

After selecting the entry Statistic from the channel submenu (see Figure 13.6, “Add/remove channel - Submenu”), the user receives a statistic channel overview .

13.4. Terrestrial

The window Terrestrial (see Figure 13.10, “Plugin BDA Tuner - Terrestrial”) contains available and selectable DVB-T regions in the upper part. If the user selects one of these regions, the corresponding channels appear in the lower part of the window.

Figure 13.10. Plugin BDA Tuner - Terrestrial


The settings will be confirmed with the Apply button.

Note

13.4.1. Add / remove region

New channel regions can be added to or removed from the region list in the upper part of the plug-in window Terrestrial. For this the right mouse button has to be used. The submenu shown in Figure 13.11, “Add / remove region - Submenu” opens.

Figure 13.11. Add / remove region - Submenu


Add region

To add a channel region, the menu entry Add Region must be selected from the region list (see Figure 13.11, “Add / remove region - Submenu”). A new line will be created in the table. The line entries can be edited by double clicking .

Remove region

To remove a channel region, the menu entry Add Region must be selected from the region list (see Figure 13.11, “Add / remove region - Submenu”). A note appears that must be confirmed by the user.

The OK button confirms the deletion. The selected channel region disappears out of the list.

The Cancel button aborts the process.

13.4.2. Add / remove transponder

A new transponder can be added to or removed from the transponder list in the lower part of the plug-in window Terrestrial. For this the right mouse button has to be used. The submenu shown in Figure 13.12, “Add / remove transponder - Submenu” opens.

Figure 13.12. Add / remove transponder - Submenu


Add trandponder

To add a transponder, the menu entry Add Transponder must been selected out of the transponder list (see Figure 13.12, “Add / remove transponder - Submenu”). A new line will be created in the table. The line entries can be edited by double clicking .

Remove transponder

To remove a transponder, the menu entry Remove Transponder must been selected out of the transponder list (see Figure 13.12, “Add / remove transponder - Submenu”). A note appears which must be confirmed by user.

The OK button confirms the deletion. The selected transponder disappears from the list.

The Cancel button aborts the process.

13.5. EPG

The window EPG shows a list of all available broadcast information of the chosen channel (see Figure 13.13, “Plugin BDA Tuner - EPG”). The currently running broadcast is shown with a green background.

Unlike the Set Top Box EPG plug-in , the EPG information will be taken out of the transponder transport stream.

Figure 13.13. Plugin BDA Tuner - EPG


Rec: Recording mark check box (see Section 13.5.1, “Recording via EPG”

Date: Date of the broadcast

Begin: Start time of the broadcast

End: End time of the broadcast

Duration: Duration of the the broadcast (in minutes)

Event: Name of the broadcast

Description: Description of broadcast

The user can receive further broadcast information by marking the respective entry. The content will be filled in the lower part of the EPG list.

Note

only information transmitted by the channel is available!

The settings will be confirmed with the Apply button.

The OK button closes the window and the Cancel button aborts the process.

Note

Not available functions are greyed out.

13.5.1. Recording via EPG

bitcontrol® Digital TV Link makes it possible to record broadcast directly via the EPG of the BDA plug-in. For this, the checkbox in front of the respective entry must be marked. Afterwards the marked broadcast appears in red colour in the list.

It is possible to mark more than one broadcast (of different channels) for recording.

The marked broadcasts appear in the scheduler list of the Recorder plug-in (see Section 19.2, “Scheduler”). The priority of overlapping broadcasts are visible and can be changed here.

Note

Recording requires a running bitcontrol® Digital TV Link!

A marked broadcast can be reset with an additional click on the check box.
