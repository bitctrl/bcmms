Note

The Server plug-in only provides information. It isn't possible to parameterize it in bitcontrol® Digital TV Link. For the adjustment of Server settings the masks of bitcontrol® Video Streaming Server (bcVSS) can be used or the included Server.xml file (see C:\ProgramData\BitCtrl\DTVLink\Plugins) can be edited by hand.

The main window of the plug-in (see Figure 20.1, “Plug-in Server - Main window”) is used to Start or Stop the Server with the respective buttons.

20.2. Statistics

The different Statistics windows contain further statistical information about Multistream Server processes.

Slider Server

The slider Server contains a table with general server parameters and their respective values.

Slider Listeners

This table contain information about the TCP/IP ports of external connections to the Multistream Server.

Slider Connections

This table shows statistical values of current connections from clients to the server.

Slider Groups and Users

The tables of the sliders Groups and Users provide information about user authentication.

Slider Live Channels

This table shows information about the channels that are available for streaming (see also OnAir option, Chapter 6, Channels).

Slider Media Sessions

The windows of the slider Media Session provide values of Multistream Server Media Sessions. Since there are different Media Sessions types each Media Sessions window is separated again, so it is possible to show statistical values for each Media Sessions type in their own slider. Here is another separation in the slider: General information about the Media Sessions type (Info) and information about the ClientSessions.
