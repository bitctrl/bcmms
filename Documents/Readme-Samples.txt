Follow the next steps to try the screencast function.

 +++++++++++++++++++++++++++++++++++++++++++++++++++++
 +
 +   ATTENTION!
 +
 +   The screencast function requires ffdshow encoder!
 + 
 +++++++++++++++++++++++++++++++++++++++++++++++++++++

*  In the taskbar, you can see two
   bitcontrol® Multimedia Suite icons.

*  Click on the left one and select
   the item Capture --> Screencast
   from the menu. 

*  SimplePlayer switches to Screencast
   and shows a part of your local desktop.

*  You can use the same procedure for
   switching back to File. To do this, select
   the menu item File --> Example video
