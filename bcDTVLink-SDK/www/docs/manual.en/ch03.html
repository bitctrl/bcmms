3.1. Licensing under Windows Vista and Windows 7

Note

bitcontrol® Digital TV Link is part of bitcontrol® Multimedia Suite. The license of bitcontrol® Multimedia Suite already contains some bitcontrol® Digital TV Link licenses (see price list on www.bitcontrol.com). Single licenses for bitcontrol® Digital TV Link are always additions to a valid bitcontrol® Multimedia Suite license (masterkey).

After installation bitcontrol® Digital TV Link runs in demo mode. That means it will stop working every 30 minutes. If the user configures bitcontrol® Digital TV Link again, it can be tried for another 30 minutes.

For a longer software test period a trial license can be requested. This will be sent by e-mail after registration on www.bitcontrol.com.

If bitcontrol® Digital TV Link achieves the requirements a full license can be bought in the online shop of www.bitcontrol.com.

The trial and demo version run without any restrictions.

License installation

The installation of a bitcontrol® multimedia product license occurs through bitcontrol® License Managers. This application will be installed automatically during bitcontrol® Multimedia Suite installation.

bitcontrol® License Manager will be started via Windows® start menu entry bitcontrol. The window of bitcontrol® License Manager opens where you can see all licenses of bitcontrol® products which are already installed (example see Figure 3.1, “Start window of bitcontrol®License Manager”).

Figure 3.1. Start window of bitcontrol®License Manager


With the button Install New the bitcontrol® Digital TV Link license can be added. For this, the user has to choose the path to the license file (*.lic) and open it. (Figure 3.2, “Path to the license file”).

Figure 3.2. Path to the license file


The bitcontrol® Digital TV Link license appears in the bitcontrol® License Manager administration window. The installation was successful (Figure 3.3, “Result of bitcontrol® Digital TV Link license installation”).

Figure 3.3. Result of bitcontrol® Digital TV Link license installation


Alternatively, the license file can be added by using drag and drop.

By selecting the checkbox in front of a license file, a license can be activated or deactivated.

Note

If a 64bit version of Windows Vista or Windows 7 is used, the described procedure should be additionally done with the bitcontrol® license manager for 64bit.

Note

You will need further licenses for the plug-ins Traficon Vip-T, Bosch Vip X, Axis Devices, Geutebrück GeViScope and RTSP Server.

3.1. Licensing under Windows Vista and Windows 7

Note

The following licensing method should be prefered under Windows Vista and Windows 7.

Open the directory C:\Programm Files\Common Files\BitCtrl\Licenses in the Windows file system. It contains all installed bitcontrol® licenses. Copy the bitcontrol® license into this directory.

If a 64bit version of Windows Vista or Windows 7 is used, it will be necessary to additionally copy the bitcontrol® license in the directory C:\Programm Files(x86)\Common Files\BitCtrl\Licenses.

Note

To do any action (like delete or copy) in the mentioned directory "Common Files Directory" of Windows Vista or Windows 7, you must have administrator rights!
