This chapter consists of an overview over the hardware and software requirements, that are necessary for the optimal use of bitcontrol® Digital TV Links.

2.1. Hardware

2.1.1. CPU und Ethernet

The software was developed to be used on PCs running the MS Windows® operating system. It is executable on nearly all commercially available x86 Intel® and compatible processors. If the video data source operates at high bit rates and the processor is set to slow clock frequencies, the software may not be able to completely receive the MPEG data in real-time. In this case the video no longer runs smoothly and may contain artifacts. Individual frames will be discarded.

Special care has to be taken when configuring the network. DVB streams are typically transmitted at 2,5 Mbit/s (DVB-T), 3,5 Mbit/s (DVB-C/S) ,9 MBit/s (DVB-S) and up to 19 Mbit/s for HDTV. We therefore recommend a 100 Mbit/s network.

If the Multicast Retransmitter option of the bitcontrol® Digital TV Link is used, it is important to make sure that the active network components (switches) support multicast mode on every port. Otherwise, the network traffic caused by the broadcast transmissions would put unnecessary load on the network. It is also safer to configure the ethernet ports for half-duplex mode. Full-duplex operation and auto-mode should be reserved for network specialists.

2.1.2. Hardware requirement for USB TV sticks and/or PCI cards

If a USB TV stick and/or a PCI card is used, a free USB slot for the stick and/or a free PCI slot for the PCI card will be needed, respectively.

Furthermore you should follow the requirements of the producer. You can find them on the product packaging or in the product documentation.

2.1.3. Hardware requirement for other video sources

If another video souce (e.g. Bosch Vip X, Traficon Vip-T, Geutebrück GeViScope, Axis devices, ...) is used, respective data links/interfaces are necessary.

Furthermore you should follow the requirements of the producer. You can find them on the product packaging or in the product documentation.

2.2. Software

2.2.1. DirectX®

For bitcontrol® Digital TV Link Microsoft® DirectX® is required. Please install the respective latest version of the DirectX® package from the Microsoft® homepage http://www.microsoft.com/downloads.

2.2.2. Software requirements for USB TV stick, PCI cards or other devices

If a USB TV stick, a PCI card or another device is used, you have to be sure, that the device is working correctly. So please follow the instructions of the device producer.

Note

Usually it is sufficient to just install the device driver!!

2.2.3. Third-party software

bitcontrol® Digital TV Link is able to integrate third-party software (e.g. encoder or other filter software) to enlarge the functionalities. If such a software should be used it is necessary to make sure that the software is working correctly.
