Chapter 10. bitcontrol® Digital TV Link control

If bitcontrol® Digital TV Link is loaded, at least one bitcontrol® Digital TV Link systray icon appears in the footline of the desktop. Depending on the configuration (e.g. started transcoder with bitcontrol® Video Decoder), further systray icons of bitcontrol® Multimedia Applications are visible. The context menu can be now opened by clicking the right mouse button (see example in Figure 10.1, “bitcontrol® Digital TV Link control via context menu (example: own channel bouquet for BDA)”).

Note

Which functions are available depend on the loaded plug-ins and the adjusted settings .

Console: If this option is active, the bitcontrol® Digital TV Links Console will be shown. More information can be found in Chapter 11, Console.

Record: Start / Stop video record

Figure 10.1. bitcontrol® Digital TV Link control via context menu (example: own channel bouquet for BDA)


Current running channel and/or channel history Show the channels in the order of their selections. The currently running channel is in first position.

Created bouquets (for BDA): If channel bouquets were created by the user, they are listed here. Each bouquet entry has its own submenu (see example in Figure 10.1, “bitcontrol® Digital TV Link control via context menu (example: own channel bouquet for BDA)”) which can be used to select an channel.

Note

This selection is only available for the BDA Tuner plug-in.

QuickSwitch (for BDA) The Quickswitch submenu can be used to select another channel from the channel bouquet.

Note

This selection is only available for the BDA Tuner plug-in.

Receiver Type: All started plug-ins are listed here. Each plug-in owns a submenu which is used to switch channels.

Audio Tracks: The language can be set via this menu entry.

Note

Only transmitted languages are available!

EPG: This menu entry opens a window with channel broadcast information. More information can be found in Section 13.5, “EPG” .

Filters: The Filter context menu (see Figure 10.2, “bitcontrol® Digital TV Link control via context menu - Filter context menu”) is used to change filter options for the selected channel.

Figure 10.2. bitcontrol® Digital TV Link control via context menu - Filter context menu


Options: This menu entry opens the bitcontrol® Digital TV Link properties window. More information can be found in Section 5.1, “Window Options”.
