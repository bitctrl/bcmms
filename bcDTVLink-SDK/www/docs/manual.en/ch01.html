Chapter 1. Introduction

The bitcontrol® Digital TV Link is a DirectShow® filter, a COM object for multimedia applications running under Microsoft Windows®. The software is no stand-alone program. It is a component loaded by multimedia players when needed.

GG: Der bitcontrol® Digital TV Link ist ein universeller Filter für Capturing, Screencast, Streaming, Recording und Transcoding von Audio/Video in einem Multimedia-Player. Die ladbare Bibliothek trägt die Dateibezeichnung The product is a streaming filter for video streams on the input of a multimedia player. The file name is bcDTVLink.dll.

The following sources can be used as video stream input:

Set Top Boxes (STB) of the type

  • DreamBox (see http://www.dream-multimedia-tv.de/)

  • d-Box II under Linux (see http://wiki.tuxbox.org/ , http://www.dbox2.info/)

  • ReelBox (see http://www.reel-multimedia.com)

DVB PCI cards and USB DVB-T sticks

  • bitcontrol® DTVLink supports BDA. So it is possible to use DVB PCI cards and USB DVB sticks from different producers (e.g. TechnoTrend, Terratec, Haupauge, Pinnacle, etc.).

    Note

    If your device doesn't work with bitcontrol® software, please contact the BitCtrl Systems GmbH support team.

Sources like

  • Capture-Device with Windows® Video-Capture-Interface Support (e.g. Webcam)

  • Streaming output (multicast/unicast) of another bitcontrol® Digital TV Link

  • Encoder devices

    • Bosch Vip X

    • Traficon Vip-T

    • Axis devices (e.g. camera)

  • Encoder devices and encoder cards

The bitcontrol® Digital TV Link supports different input formats depending on the video source:

  • PES (packetised elementary video streams)

  • TS (transport stream)

  • RTP/H.263 (realtime protocol)

  • RTSP/RTP MPEG-4 (realtime streaming protocol)

  • bcmsdb (unicast / multicast)

  • *.mpg file (m1v, m2v, m2p, mpa, mpeg, mp2)

  • *.vob file

  • *.ts file

  • *.avi file (MPEG-4 part 2)

  • *.mp4 (in part DivX)

Additionally, bitcontrol® Digital TV Link is able to retransmit the input video stream as unicast, multiple-unicast or multicast stream into the network, effectively making it a video server.

Furthermore it is possible to record the received video signal directly onto hard disk. The file can be played-back afterwards in Windows Media® player or loaded into a video editing program.

The bitcontrol® Digital TV Link filter is useable in combination with other bitcontrol® multimedia products as well as in other multimedia players (if these programs can load DirectShow® filters).

The bitcontrol® Digital TV Link is characterized by its high robustness and high quality, the latter being achieved through absolutely correct color decoding with 4:2:2 Up-sampling without any color artifacts as well as through correct black-white decoding. The bcDTVLink can handle HDTV videos of all formats up to a resolution of 1080p (1920x1080 progressive) at a data rate of up to 80 Mbit/s. It can be used in the private Home theatre PC-HTPC as well as in industrial products.

Note

For optimum performance the usage of bitcontrol® Video Decoder is strongly recommended.
