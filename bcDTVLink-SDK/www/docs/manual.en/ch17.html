The user receives a window where he can create, see and administrate a list of video files in different formats (see Figure 17.1, “Table in plug-in File (example)”) via the menu entry File. The playback of a video file from that list occurs via the bitcontrol® Digital TV link menu (see Chapter 10, bitcontrol® Digital TV Link control), which can be opened by clicking the right mouse button over the systray icon.

Figure 17.1. Table in plug-in File (example)


The following video file properties are available in the table:

Name: Name of the video file

Path: Path of the storage location of the video file

Enable: This options shows whether a video is played during video sequence playback (all videos of the table are played back in a sequence) (yes) or not (no).

Duration: This field shows how long a video is already running in seconds.

Note

During the first start of this option directly after the installation, the window is still empty.

The plug-in File has a context menu (see Figure 17.2, “Plug-in File context menu”) that can be opened by right clicking in the window.

Figure 17.2. Plug-in File context menu


The context menu provides the following functions:

Add File(s): Adds a video file to the table (see Section 17.1, “Adding video files to a playlist”)

Remove: Removes the video file from the table.

Enable: Set Enable option to yes. It means the video will be played back.

Disable: Set Enable option to no. It means the video will be not played back.

Move Up: Moves the marked table entry one position up, so the video sequence changes.

Move Down: Moves the marked table entry one position down, so the video sequence changes.

17.1. Adding video files to a playlist

The user selects the entry Add File(s) fromthe submenu. The typical Windows® search dialogue opens. The respective video will be selected there.

If the respective video file was found, it can be added to the playlist with the Open button.

Figure 17.3. Video file assumed in playlist


The play list entries will be assumed with the Apply button. The entry colour changes from red to black. The video files are ready for playback.

17.2. Video file playback from the playlist

To playback a video file from the playlist, the user clicks the right mouse button on the bitcontrol® Digital TV Link tray icon in the footline. A menu opens. The user selects the entry File. Another submenu opens that contains the play list entries. The video file that should be played back can be chosen here.

Figure 17.4. Open video file via bitcontrol® Digital TV Link tray icon menu


17.3. Delete video file from the playlist

To delete a video file from the playlist, the user has to mark the respective entry and click the right mouse button. A submenu (siehe Figure 17.2, “Plug-in File context menu”) appears. The user selects the entry Remove there and the video file will be deleted from the playlist.
