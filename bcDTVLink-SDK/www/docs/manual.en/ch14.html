14.1. Options

Note

This plug-in was created for industrial purposes.

Bosch VIPX encoder devices are supported as input sources by bitcontrol® Digital TV Link via the plug-in BoschVIPX.

The plug-in can be started in the main windows (see Figure 14.1, “Plug-in BoschVIPX - Main window”) with the help of the Start button and can be stopped with the Stop button.

Figure 14.1. Plug-in BoschVIPX - Main window


14.1. Options

The window Options of BoschVIPX plug-in (see Figure 14.2, “Plug-in BoschVIPX - Option window”) can be used to set the properties of Bosch encoder.

Figure 14.2. Plug-in BoschVIPX - Option window


Enable Multicast Loopback: This Option allows the reception of locally sent video streams.

Address: The value specifies the IP address (> 224.x.x.x) of the input and/or the output stream.

Bind Address: Indicates, on which network card the multistream should be received. The choices are: Auto (standard network card), Any (every network card) and Loopback (broadcast and/or local computer).

Receive Buffer Size (KB): The value defines the maxium buffer size for send-site data packages.

Statistics: The field Statistic contains statistical values such as received data packets (total), amount of received data packets in bytes (total), etc.

The settings will be confirmed with the Apply button.

The OK button closes the window and the Cancel button aborts the process.
