Chapter 11. Console

A Console window is available to visualize and document bitcontrol® Digital TV Link processes. It can be opened by activating the option box Show Console in bitcontrol® Digital TV Link Preferences dialogue (see Section 5.3, “Window Preferences”).

Figure 11.1. Console window


Verbosity Level: The information amount can be regulated by the slider Verbosity Level. The levels 0 (no information) up to 15 (all available information) are possible.

Filter: The console window messages can be classified via the selection field Filter. The following possibilities are selectable: All Source (all messages), DTVLink, BDA Tuner, DBox2, DreamBox, ReelBox, AXIS, File, Geutebrück GeViScope, RTSP Receiver and TraficonVIPT. The default value is All Sources.

Topmost: Activating the Topmost option results in the Console window always laying on top of all other windows on your display. This option is not active by default.

Last on Top: If the option Last on Top is active, the last console message will be shown in the first place. This option is not active by default.

Auto Refresh: The option Auto Refresh allows an automatic console window updating. This option is active by default.
