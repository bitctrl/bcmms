Chapter 18. Plug-in NetPublisher

NetPublisher is the client input plug-in for the Net Receiver Media Session of the Multistream Servers.

A NetPublisher client must be configured for a special Net Receiver Media Session, but a Net Receiver Media Session is able to receive the video streams of several clients.

Note

The video stream transmission from NetPublisher to Multistream Server occurs via HTTP. This allows local safety barriers (firewall, packet filter, etc.) to be bypassed.

Figure 18.1. Interaction of NetPublisher and Net Receiver Media Session


bitcontrol® Digital TV Link provides the following parameters for NetPublisher plug-in configuration (Figure 18.1, “Interaction of NetPublisher and Net Receiver Media Session ”).

Address*: This field contains the IP address and port (example 192.x.x.x:8080) of the Multistream Server.

Session name: Name of the Net Receiver Media Session for which the NetPublisher is configured as input client.

Source name: Name of the connected video source. The name can be freely allocated.

Stream Name: Name of the data stream.

The settings will be assumed with the Apply button.

Note

Further information about NetPublisher can be found in the bitcontrol® Video Streaming Server user manual.
